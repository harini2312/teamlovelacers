1. First, you will see the folders backend, web-frontend, and mobile-frontend
2. Install all requirements for the web-frontend, and the same goes for the mobile-frontend
3. Run the backend by entering the root folder and check with the api checkpoints by creating users, posting posts, and notifications, and checking them in real-time in the frontend.
4. Install beforehand while testing in the mobile frontend the Expo Go app, and then scan the QR code in the terminal when you run the mobile-frontend to see the UI in the mobile interface.
5. That's it !!
